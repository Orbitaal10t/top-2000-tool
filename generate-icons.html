Icon Generator

Genereer alle benodigde app icons voor de Top 2000 Tool

Instructies

  1. Klik op "Download Alle Icons" om alle icon formaten te downloaden
  2. Of download individuele icons met de knoppen onder elke preview
  3. Plaats alle gedownloade icons in de icons/ map van je project
  4. De bestanden hebben de juiste namen: icon-72.png, icon-96.png, etc.
  5. Commit en push de icons naar je repository
  6. Klaar! Je app kan nu op het homescherm worden geplaatst
